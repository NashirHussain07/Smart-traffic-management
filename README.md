### 🚀 Executive Summary

Modern urban infrastructure relies on outdated, static timing mechanisms that create catastrophic bottlenecks for medical emergencies. This project introduces a highly reactive **Intelligent Traffic Management System**. Utilizing real-time video feeds, the architecture autonomously detects approaching emergency vehicles and triggers preemptive signal overrides, effectively removing human dispatch from the critical path and accelerating emergency response times.

### ⚙️ System Architecture

* **Edge AI Engine:** Powered by a heavily optimized **YOLOv8** model for high-speed, high-accuracy object detection under diverse environmental conditions.
* **Real-Time Frame Processing:** Utilizes **OpenCV** to stream and analyze intersection camera data with sub-second latency.
* **Algorithmic Grid Override:** A deterministic logic controller that calculates vehicle trajectory and dynamically manipulates traffic light phases to clear intersection bottlenecks before the ambulance arrives.

```bash
